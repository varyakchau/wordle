## Описание проекта

Проект представляет собой веб-игру по мотивам Wordle. Пользователь выбирает тему и длину слова, после чего должен угадать гуманитарный термин за ограниченное количество попыток.Полезна для расширения кругозора или подготовки к гуманитарным экзаменам! Написана на питоне с помощью django, фронтенд на html. 

## Структура репозитория

- config/ - настройки всего Django-проекта
- wordle/ - основное приложение с игрой
- wordle/data/ - файлы с разрешёнными словами и терминами
- wordle/management/ - команды Django, например, загрузка слов в базу
- wordle/migrations/ - файлы изменений базы данных
- wordle/templates/ - HTML-страницы сайта
- wordle/tests/ - есты проекта
- wordle/models.py - описание данных
- wordle/views.py - основная логика сайта: регистрация, игра, рейтинг
- wordle/urls.py - адреса страниц и запросов
- wordle/utils.py - вспомогательные функции для проверки слов
- manage.py - файл для запуска команд Django
- Dockerfile - инструкция для сборки контейнера с сайтом
- docker-compose.yml - запуск сайта и базы данных вместе
- pyproject.toml - зависимости проекта 
- uv.lock - зафиксированные версии зависимостей
- README.md - описание проекта

## Запустите проект вот так:

```
docker compose up -d
```

После запуска приложение будет доступно по адресу:

http://localhost:8000
